## App Description

This application is a native Android app developed using Kotlin in Android Studio.

The purpose of the app is to help users distinguish between real-life productivity hacks and common urban myths. The app presents a series of flashcard-style questions where users must decide whether each statement is a Life Hack (True) or an Urban Myth (False).

## Features

- Welcome screen with app introduction
- Flashcard quiz with multiple questions
- True/False answer selection
- Instant feedback after each answer
- Score tracking system
- Final score screen with personalised feedback
- Review screen showing questions, answers, and explanations

## Application Logic

- Arrays are used to store questions, answers, and explanations
- A loop is used to iterate through each question
- A score counter tracks correct answers
- Multiple activities are used to manage navigation between screens

## Screens in the App

- Welcome Screen
- Quiz Screen
- Score Screen
- Review Screen

## Technologies Used

- Kotlin
- Android Studio
- XML Layouts

## How to Run the App

1. Open the project in Android Studio
2. Allow Gradle to sync
3. Run the app using an emulator or physical device
